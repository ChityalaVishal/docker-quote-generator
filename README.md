    Motivation Hub 🚀

A modern Dockerized motivational web platform built using Flask, HTML, CSS, and Docker.
This project provides users with motivational quotes, inspiring wallpapers, and a thought-sharing feature through an attractive and interactive web interface.

    ✨ Features
🏠 Beautiful Welcome Home Page
📌 Sidebar Navigation Dashboard
✨ Random Quote Generator
🖼 Motivation Image Gallery
✍ User Thought Sharing System
🎨 Modern Responsive UI
🐳 Dockerized Deployment
⚡ Flask Backend Integration
🛠 Technologies Used
Python
Flask
HTML5
CSS3
Docker

    🚀 How to Run the Project
1️⃣ Clone Repository
git clone https://github.com/YOUR_USERNAME/motivation-hub.git
2️⃣ Open Project Folder
cd motivation-hub
3️⃣ Build Docker Image
docker build -t motivation-hub .
4️⃣ Run Docker Container
docker run -p 5000:5000 motivation-hub
5️⃣ Open in Browser
http://localhost:5000
